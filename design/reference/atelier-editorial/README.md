# Atelier Editorial Reference

This folder freezes the selected visual source and the pre-v4 desktop baseline.

- `atelier-editorial-night.png`: selected Night visual target.
- `atelier-editorial-light.png`: Light visual target produced during A6.
- `desktop-priority.html`, `.css`, `.js`: interactive source concept.
- `assets/avatar_3d_stage_clean.png`: visual asset required by the concept.
- `v2-wpf-current.png`: desktop appearance before the v4 migration.

The concept is visual evidence, not the color source of truth. Production values live in `design/tokens.json`.

Until the A6 Light capture exists, its missing file is an explicit acceptance gap rather than an inferred pass.
